# Mission

Build an operator-trustworthy AI runtime that can be installed, configured, and
run without depending on tribal memory.

Milady should let an operator:

- install the system
- configure providers, plugins, and knowledge
- understand what the runtime is allowed to do
- recover from common failures
- answer boundary and ownership questions from canonical sources

The documentation standard follows the same rule:

proof first, operator truth first, public narrative second.
