# Docs

Milady is a local-first AI assistant that runs on your own machine. You install it, pick a character, give it a brain (local model or cloud API), and talk to it. It can also connect out to Discord, Telegram, iMessage, and other places — so the same agent you chat with on your desktop can also answer messages on your behalf somewhere else.

These are the **consumer docs**. They're written for people who downloaded the app and want to actually use it. No code. No terminal commands unless they're genuinely useful. No assumptions about what you already know.

If you're a developer building on Milady — writing plugins, hitting the REST API, using the CLI, embedding the runtime in something else — you want [docs.milady.ai](https://docs.milady.ai) instead. That's the full technical reference.

## Where to start

Pick the tier below that matches where you are. Each tier is independent — you can start at Beginner and work through, or jump straight to Advanced if you already have the app running. Nothing in later tiers assumes you read earlier ones word-for-word.

Every page tells you what it assumes, what you'll learn, and what to do next.
