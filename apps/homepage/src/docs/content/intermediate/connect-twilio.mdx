# Connect to Twilio (SMS + Voice)

Twilio gives your Milady agent a phone number. People text the number, the agent replies. People call the number, the agent picks up and has a voice conversation. Twilio is paid (cheap, but not free), very reliable, and globally available.

**What you'll learn:** sign up for Twilio, buy a phone number, grab the three credentials, set up the webhook, paste everything into Milady.

## What you need before you start

- **A Twilio account** at <a href="https://www.twilio.com/try-twilio">twilio.com/try-twilio</a>. Free trial gives ~$15 in credit, enough to buy a US number and test for a few days.
- **A payment method** — Twilio doesn't charge during the trial but requires a card for anything beyond.
- **A publicly reachable HTTPS URL** for Milady. Twilio sends incoming messages and calls via webhook. Use a tunnel (Cloudflare Tunnel, ngrok) during development.
- **Milady running** with a working provider.

## Step 1 — Sign up and verify

<Steps>
  <li>Open <a href="https://www.twilio.com/try-twilio">twilio.com/try-twilio</a> and create an account.</li>
  <li>Verify your own phone number — Twilio sends a test code.</li>
  <li>Answer the "what are you building" questions honestly; they affect which features Twilio enables by default.</li>
</Steps>

## Step 2 — Grab the Account SID and Auth Token

<Steps>
  <li>Open the Twilio Console at <a href="https://console.twilio.com">console.twilio.com</a>.</li>
  <li>On the main dashboard, find <strong>Account SID</strong> (starts with <code>AC</code>) and <strong>Auth Token</strong>. Copy both.</li>
</Steps>

<Callout kind="warning">
The Auth Token is a master credential — it can send messages, make calls, buy more numbers, and rack up charges. Treat it like a password. Rotate it in Console → Settings → General if it ever leaks.
</Callout>

## Step 3 — Buy a phone number

<Steps>
  <li>In the Twilio Console, go to <strong>Phone Numbers → Manage → Buy a number</strong>.</li>
  <li>Pick a country. Enable <strong>SMS</strong> and <strong>Voice</strong> capabilities (tick the boxes).</li>
  <li>Pick a number and buy it. US local numbers are ~$1/month.</li>
  <li>Copy the number in E.164 format (e.g. <code>+15551234567</code>).</li>
</Steps>

## Step 4 — Set up the messaging webhook

<Steps>
  <li>In the Console, open the phone number you just bought.</li>
  <li>Under <strong>Messaging</strong> → <strong>A MESSAGE COMES IN</strong>, set Webhook to <code>https://&lt;your-public-milady-url&gt;/webhooks/twilio/sms</code>.</li>
  <li>Method: <code>HTTP POST</code>.</li>
  <li>Save.</li>
</Steps>

For voice calls, also set:

<Steps>
  <li>Under <strong>Voice &amp; Fax</strong> → <strong>A CALL COMES IN</strong>, set Webhook to <code>https://&lt;your-public-milady-url&gt;/webhooks/twilio/voice</code>.</li>
  <li>Save.</li>
</Steps>

## Step 5 — Hand credentials to Milady

<Steps>
  <li>Open Milady. Go to <strong>Settings → Plugins → Twilio → Configure</strong>.</li>
  <li>Paste the <strong>Account SID</strong>.</li>
  <li>Paste the <strong>Auth Token</strong>.</li>
  <li>Paste the <strong>Phone number</strong> in E.164 format.</li>
  <li>(If running Milady locally behind a tunnel) Set <strong>Webhook URL</strong> to your public URL base so Milady's self-references match.</li>
  <li>Click <strong>Save</strong>.</li>
</Steps>

## Step 6 — Test SMS

Send a text from your personal phone to the Twilio number. Within a few seconds, Milady should respond via SMS. Check Twilio's Console → Logs → Messaging if you don't see anything — Twilio logs every inbound and outbound message with delivery status.

## Step 7 — Test voice (optional)

Call the Twilio number. Milady's voice connector answers and starts a conversation using your configured voice/TTS stack (see [Voice, talk mode, and TTS](/docs/intermediate/voice-and-tts)).

## Useful options

| Option | What it does |
| --- | --- |
| **Voice call provider** | Which voice backend handles the call. Default `twilio`. |
| **Voice call from number** | Outbound caller ID for when the agent initiates calls. |
| **Voice call inbound policy** | `allow-all`, `allow-from`, or `deny-all`. |
| **Voice call inbound greeting** | Text the agent speaks when picking up a call. |
| **Voice call max duration** | Hard cap in seconds. Default 3600 (one hour). |

## Cost reality

Twilio is pay-as-you-go. As of writing, rough US pricing:

| Action | Cost |
| --- | --- |
| Phone number (monthly) | ~$1 |
| Incoming SMS | ~$0.0075 per message |
| Outgoing SMS | ~$0.0079 per message |
| Incoming voice call | ~$0.0085 per minute |
| Outgoing voice call | ~$0.014 per minute |

Light personal use costs a few dollars a month. Business use can scale into real money — set billing alerts.

## Troubleshooting

**SMS doesn't arrive at Milady.**
Check Twilio Console → Logs → Messaging. Twilio logs every attempt with the HTTP response from your webhook. If it shows a timeout or error, your public URL is wrong or unreachable.

**Milady receives the message but replies never arrive at the phone.**
Auth token is wrong. Re-copy from the Console dashboard.

**Voice call connects but audio is one-way or silent.**
Your voice/TTS pipeline in Milady isn't configured. See [Voice, talk mode, and TTS](/docs/intermediate/voice-and-tts).

**International SMS fails silently.**
Some country codes require Twilio to approve you first (anti-fraud). Check Console → Messaging → Settings → Geo permissions.
