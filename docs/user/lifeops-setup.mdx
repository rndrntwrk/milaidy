LifeOps is the Milady surface for tasks, habits, routines, reminders, and goals.

## Open LifeOps

If you do not see LifeOps in the app, update Milady first.

## Start with one item

Create one useful item first:

- `task` for one-offs
- `habit` for recurring behavior
- `routine` for a recurring pattern

Examples:

- `Take meds at 8am`
- `Inbox zero before lunch`
- `Stretch for 10 minutes every evening`

## Configure the basics

For each item, set:

1. title
2. schedule
3. priority
4. reminders

## Add a goal if needed

Use goals when several items support the same outcome.

Examples:

- `Sleep better`
- `Ship the launch`
- `Stay on top of email`

## Optional Google connection

Connect Google if you want:

- calendar awareness
- Gmail triage or reply drafting
- reminders that fit around your schedule

## Daily actions

- complete
- snooze
- skip

## Common problems

### Reminders do not fire

Check notification permissions.

### Google context does not appear

Reconnect the Google integration and confirm the correct account is linked.

### LifeOps is missing

Update Milady.
