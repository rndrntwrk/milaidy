Los docs de Milady cubren el uso del producto. Los docs de elizaOS cubren el framework y el trabajo de construcción.

<div id="milady-docs">

## Docs de Milady

</div>

- configuración de proveedores
- cambios de personaje
- capacidades instaladas
- inicio y control de la aplicación
- conexiones remotas
- configuración de conectores (Discord, Telegram, Slack, WhatsApp, Signal, iMessage y [22 más](/es/guides/connectors))
- LifeOps
- solución de problemas

<div id="elizaos-docs">

## Docs de elizaOS

</div>

- `app-core`, `ui` y `agent`
- internos del runtime
- agentes, memoria, proveedores y servicios
- creación de plugins
- referencia REST
- guías de despliegue
