# `milady plugins`

Manage the plugin lifecycle — browse the registry, install or remove plugins, and inspect what's currently loaded.

## Subcommands

| Subcommand | Purpose |
|------------|---------|
| `milady plugins list` | Browse available plugins in the registry |
| `milady plugins installed` | Show plugins currently installed |
| `milady plugins install <name>` | Install a plugin by name or package |
| `milady plugins uninstall <name>` | Remove an installed plugin |
| `milady plugins search <query>` | Search registry plugins by keyword |

## Subcommands

### `milady plugins list`

List all plugins from the registry. Shows install status and basic metadata.

| Flag | Description |
|------|-------------|
| `-q, --query <query>` | Filter plugins by name or keyword |
| `-l, --limit <number>` | Max results to show (default: 30) |

```bash
milady plugins list
milady plugins installed
milady plugins install @elizaos/plugin-openai
milady plugins install discord
milady plugins uninstall discord
milady plugins search "image generation"
```

Short plugin names (e.g. `discord`) resolve to `@elizaos/plugin-discord` automatically. Full package names also work.
