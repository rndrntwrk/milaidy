# `milady doctor`

Runs health checks across four categories: System, Configuration, Storage, and Network. Reports pass/fail/warn status for each check with suggested fixes.

## Options

| Flag | Description |
|------|-------------|
| `--no-ports` | Skip port availability checks |
| `--fix` | Automatically fix issues where possible (runs `milady` sub-commands for auto-fixable checks) |
| `--json` | Output results as JSON (CI-friendly). Exits with code 1 if any checks fail. |

## Examples

```bash
milady doctor
milady doctor --fix
milady doctor --json
milady doctor --no-ports
```

Use the output when filing issues or after moving state directories.
