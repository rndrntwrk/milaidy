# `milady dashboard`

Opens the web dashboard in your default browser, or prints its URL if a browser cannot be launched.

## Examples

```bash
# Open dashboard in browser
milady dashboard

# Show available flags (URL-only mode, etc.)
milady dashboard --help
```

## Dashboard ports

The dashboard URL depends on your port configuration:

| Mode | Default URL |
|------|------------|
| Production | `http://localhost:2138` (`MILADY_PORT`) |
| Dev mode | `http://localhost:2138` (Vite dev server on `MILADY_PORT`, API on `MILADY_API_PORT`) |

Override with environment variables:

```bash
MILADY_PORT=3000 milady start
# Dashboard at http://localhost:3000
```
