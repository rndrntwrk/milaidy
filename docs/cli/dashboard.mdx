# `milady dashboard`

Opens the Control UI in your default browser. Checks ports in order: the specified `--port` (default 2138), then the dev server port. If neither is running, starts a Vite dev server for the app UI.

## Flags

| Flag | Description |
|------|-------------|
| `--port <port>` | Server port to check (default: `2138`). |
| `--url <url>` | Explicit URL to open (overrides port detection). |

## Examples

```bash
milady dashboard
milady dashboard --port 3000
milady dashboard --url http://localhost:8080
```
