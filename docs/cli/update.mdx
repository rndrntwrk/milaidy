# `milady update`

Checks for updates on the current release channel and installs them. Supports switching between release channels (stable, beta, nightly).

## Flags

| Flag | Description |
|------|-------------|
| `-c, --channel <channel>` | Switch release channel before updating (`stable`, `beta`, `nightly`). |
| `--check` | Check for updates without installing. |
| `--force` | Force update check (bypass interval cache). |

## Subcommands

| Subcommand | Description |
|------------|-------------|
| `milady update status` | Show current version and available updates across all channels. |
| `milady update channel [channel]` | View or change the release channel. |

## Examples

```bash
milady update
milady update --check
milady update --channel beta
milady update --force
milady update status
milady update channel
milady update channel nightly
```
