# `milady update`

Checks for a newer Milady release and updates based on how you installed (npm global, Homebrew, desktop app, git checkout).

## Examples

```bash
# Check for updates and apply
milady update

# Show available flags
milady update --help
```

## Update channels

Configure which release channel to follow in `~/.milady/milady.json`:

```json5
{
  update: {
    channel: "stable",          // "stable" | "beta" | "nightly"
    checkOnStart: true,         // check for updates when milady starts
    checkIntervalSeconds: 14400, // 4 hours between checks
  },
}
```

## Desktop app

The desktop app checks for updates automatically on launch. Updates are downloaded in the background and applied on next restart.
