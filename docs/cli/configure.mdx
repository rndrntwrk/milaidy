# `milady configure`

Prints configuration guidance: how to read and set config values, and the most common environment variables for model providers.

This command does not accept flags. For reading and writing config values, use [`milady config`](/cli/config). For interactive key setup, use [`milady setup`](/cli/setup).

## Output

```
Set values with:
  milady config get <key>     Read a config value
  Edit ~/.milady/milady.json directly for full control.

Common environment variables:
  ANTHROPIC_API_KEY    Anthropic (Claude)
  OPENAI_API_KEY       OpenAI (GPT)
  AI_GATEWAY_API_KEY   Vercel AI Gateway
  GOOGLE_API_KEY       Google (Gemini)
```
