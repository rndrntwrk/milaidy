# `milady models`

List available models and active providers in the current configuration.

## Examples

```bash
# List all providers and their status
milady models

# Show help and available flags
milady models --help
```

## What it shows

- Each provider plugin that is auto-enabled or manually configured
- Whether the provider's API key is set
- The active model selection from `agents.defaults.model`

## Setting the default model

Edit `~/.milady/milady.json`:

```json5
{
  agents: {
    defaults: {
      model: {
        primary: "anthropic/claude-sonnet-4.6",
        fallbacks: ["openai/gpt-4o"],
      },
    },
  },
}
```

Or switch mid-session using the `/model` chat command:

```
/model openai/gpt-4o
```
