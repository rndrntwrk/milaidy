# `milady models`

Lists all supported model providers and whether each has an API key or URL configured in the environment.

Checked providers: Anthropic (Claude), OpenAI (GPT), Vercel AI Gateway, Google (Gemini), Google Vertex AI, Groq, xAI (Grok), OpenRouter, DeepSeek, Together AI, Mistral, Cohere, Perplexity, Zai, Ollama (local), elizaOS Cloud.

## Examples

```bash
milady models
```

```
Model providers:
  Anthropic (Claude): configured
  OpenAI (GPT): not set
  Ollama (local): configured
  ...
```
