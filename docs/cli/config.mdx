# `milady config`

Config helpers for reading and inspecting the active configuration.

## Subcommands

| Subcommand | Description |
|------------|-------------|
| `milady config get <key>` | Read a config value by dot-notation key (e.g. `env.ANTHROPIC_API_KEY`). |
| `milady config path` | Print the resolved config file path. |
| `milady config show` | Display all configuration values grouped by section. |

### `milady config show` flags

| Flag | Description |
|------|-------------|
| `-a, --all` | Include advanced/hidden fields. |
| `--json` | Output as raw JSON. |

## Examples

```bash
milady config get env.ANTHROPIC_API_KEY
milady config path
milady config show
milady config show --json
milady config show --all
```
