# `milady config`

Read-only helpers for inspecting the active configuration.

## Subcommands

| Subcommand | Description |
|------------|-------------|
| `milady config get <key>` | Read a specific config value |
| `milady config path` | Print the resolved config file path |
| `milady config show` | Display all config values grouped by section |

## Examples

```bash
milady config path                  # show which file the runtime loads
milady config get connectors.discord.token   # read a specific key
milady config show                  # dump all config values
milady config --help                # show available subcommands
```

Use this when you need to confirm which file path and keys the runtime will load before starting the server.
