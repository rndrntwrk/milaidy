# `milady config`

Read-only helpers for inspecting the active configuration.

## Subcommands

| Subcommand | Description |
|------------|-------------|
| `milady config get <key>` | Read a specific config key (dot-notation) |
| `milady config path` | Print the path to the active `milady.json` |
| `milady config show` | Display the resolved configuration |
| `milady config show --all` | Include advanced/hidden fields |
| `milady config show --json` | Raw JSON output |

## Examples

```bash
milady config get agents.defaults.model
milady config get gateway.port
milady config path
milady config show
milady config show --all
milady config --help
```
