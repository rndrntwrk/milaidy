# `milady db`

Low-level database utilities for the embedded PGlite database — backup, migration status, and repair.

## Examples

```bash
# Show database status and available subcommands
milady db

# Show available flags
milady db --help
```

## Database location

By default, PGlite stores data at:

```
~/.milady/workspace/.eliza/.elizadb
```

Override via config:

```json5
{
  database: {
    provider: "pglite",
    pglite: {
      dataDir: "~/.milady/workspace/.eliza/.elizadb",
    },
  },
}
```

<Warning>
Avoid running destructive database operations while the API server holds the database lock. Stop the server first with `milady stop` or `Ctrl+C`.
</Warning>

## Database security tests

Run the database security and readonly test suite:

```bash
bun run db:check
```
