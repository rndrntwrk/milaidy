# `milady db`

Database management commands.

## Subcommands

| Subcommand | Description |
|------------|-------------|
| `milady db reset` | Delete the local agent database. It will be re-created on next start. |

### `milady db reset` flags

| Flag | Description |
|------|-------------|
| `--yes` | Skip the confirmation prompt. |

The database is located at `~/.milady/workspace/.eliza/.elizadb` (or under the path configured by `MILADY_STATE_DIR` / `ELIZA_STATE_DIR`). All agent memory and conversation history will be lost on reset.

## Examples

```bash
milady db reset
milady db reset --yes
```

Avoid running destructive operations while the API server holds the database lock.
