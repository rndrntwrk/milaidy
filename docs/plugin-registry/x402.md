</Warning>

The x402 plugin implements the HTTP 402 Payment Required protocol, enabling Milady agents to handle micropayments for premium services and content access.

**Package:** `@elizaos/plugin-x402` (not yet published)

> **Note:** This plugin is an upstream elizaOS feature plugin and is not included in the bundled `plugins.json` registry. It is installable from the remote elizaOS plugin registry and auto-enables via the `features.x402` config flag.

## Installation

```bash
milady plugins install x402
```

## Enable via Features

```json
{
  "features": {
    "x402": true
  }
}
```

## Features

- HTTP 402-based micropayment handling
- Pay-per-request agent service monetization
- Integrates with blockchain wallet for payment processing
