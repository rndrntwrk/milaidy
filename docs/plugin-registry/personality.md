Enable Milady agents to evolve their personality and character traits over time.

**Package:** `@elizaos/plugin-personality`

## Overview

The Personality plugin provides elizaOS agents with self-modification and character evolution capabilities. Agents can adapt their personality traits, communication style, and behavioral patterns based on interactions and feedback. This allows agents to develop a more natural and personalized presence over time rather than remaining static.

## Installation

```bash
milady plugins install personality
```

## Configuration

This plugin has no configuration parameters. It works out of the box once installed.
