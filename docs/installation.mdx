Milady supports desktop installs and a CLI runtime. Both use the same local state directory.

## Prerequisites

| Requirement | Version | Check |
|-------------|---------|-------|
| Node.js | >= 22 | `node --version` |

The CLI installer checks for Node.js >= 22 and offers to install it if missing. The desktop app bundles its own runtime — no prerequisites needed.

## Desktop app

Download the latest stable release from [GitHub Releases](https://github.com/milady-ai/milady/releases/latest).

| Platform | File |
|----------|------|
| macOS (Apple Silicon) | `Milady-arm64.dmg` |
| macOS (Intel) | `Milady-x64.dmg` |
| Windows | `Milady-Setup.exe` |
| Linux | `Milady.AppImage` or `.deb` |
| iOS | App Store |
| Android | Google Play or APK |

### macOS

1. Download the `.dmg`.
2. Drag `Milady.app` into `Applications`.
3. Open it and continue to [First Launch](/quickstart).

### Windows

1. Download `Milady-Setup.exe`.
2. Run the installer.
3. Open Milady from the Start menu.

### Linux

1. Download the `.AppImage` or `.deb`.
2. For AppImage, run `chmod +x Milady.AppImage`.
3. Open the app and continue to [First Launch](/quickstart).

<Info>
Use the official release page. If the app looks unsigned or unfamiliar, delete it and download it again from the project release page.
</Info>

## CLI install

### macOS / Linux

```bash
curl -fsSL https://get.milady.ai | bash
milady setup
```

### Windows PowerShell

```powershell
irm https://get.milady.ai/install.ps1 | iex
milady setup
```

### Start the runtime

```bash
milady start
```

To stop the runtime, press `Ctrl+C` in the terminal or close the desktop app.

## Local state

Milady stores data in:

- macOS / Linux: `~/.milady/`
- Windows: `%USERPROFILE%\\.milady\\`

This directory holds config, conversations, uploaded knowledge, logs, and local assets.

## Alternative CLI install methods

| Method | Command |
|--------|---------|
| npm global | `npm install -g miladyai && milady setup` |
| Homebrew (macOS / Linux) | `brew tap milady-ai/milady && brew install milady` |
| Snap (Linux) | `sudo snap install milady` |
| Flatpak (Linux) | `flatpak install flathub ai.milady.Milady` |

See the [README](https://github.com/milady-ai/milady#alternative-install-paths) for the full list including APT repos and Homebrew cask.

## Before first launch

Have one of these ready:

- an API key from a supported provider (Anthropic, OpenAI, OpenRouter, Groq, etc.)
- a local Ollama model running at `http://127.0.0.1:11434`
- an Eliza Cloud account
